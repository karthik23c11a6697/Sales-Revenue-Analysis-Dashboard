# 📊 Sales & Revenue Analysis Dashboard

## 🎯 Project Overview
A complete interactive Sales & Revenue Analysis Dashboard built in Microsoft Excel, analyzing **300 transactions** across **10 products** and **5 regional markets** for FY 2024.

This project was completed as **Task 1** of the internship program.

---

## 📁 Project Files

| File | Description |
|------|-------------|
| `Sales_Revenue_Dashboard.xlsx` | Main interactive Excel dashboard with charts & KPIs |
| `Sales_Revenue_Report.pdf` | Business analysis report with insights & recommendations |
| `README.md` | Project documentation |

---

## ✅ Features Implemented

### 📈 KPI Cards (Dashboard Sheet)
| KPI | Value |
|-----|-------|
| Total Revenue | ₹1,84,32,150 |
| Total Units Sold | 7,842 |
| Total Profit | ₹50,38,920 |
| Average Order Value | ₹61,440 |
| Total Orders | 300 |

### 📊 Charts & Visualizations
- **Monthly Revenue Trend** – Line chart showing revenue across 12 months
- **Top Products by Revenue** – Horizontal bar chart ranking all 10 products
- **Revenue by Region** – Pie chart with 5 regional breakdowns
- **Monthly Profit** – Column chart showing profit trends

### 📋 Data Sheets
- `Sales_Data` – 300 raw transaction records with Order ID, Date, Product, Category, Region, Units, Price, Revenue, Margin
- `Monthly_Summary` – Aggregated monthly KPIs
- `Product_Summary` – Product-wise performance metrics
- `Region_Summary` – Regional sales breakdown with market share

---

## 🛠️ Tools Used
- **Microsoft Excel** – Dashboard, charts, KPIs, formulas
- **Dataset** – Synthetic FY 2024 sales data (300 records)

---

## 📂 How to Use

1. Download `Sales_Revenue_Dashboard.xlsx`
2. Open in Microsoft Excel (2016 or later recommended)
3. Navigate to the **DASHBOARD** tab (first tab, highlighted in orange)
4. Use the other sheets (`Sales_Data`, `Monthly_Summary`, `Product_Summary`, `Region_Summary`) to explore raw data
5. All KPIs auto-calculate from the underlying data using Excel formulas

---

## 💡 Key Insights
- **Q4** is the strongest quarter — October–December drives peak revenue
- **Laptops & Smartphones** contribute ~45% of total revenue
- **North region** leads with 20.4% market share
- **Central region** has the most growth potential
